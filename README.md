# move-audit-backend
The Multi-agent platform backend for sui move contract audit.


# Dependence

### Debian/Ubuntu
```
apt install weasyprint
```

### macOS:
```
brew install weasyprint

export DYLD_FALLBACK_LIBRARY_PATH=/opt/homebrew/lib:$DYLD_FALLBACK_LIBRARY_PATH
```
